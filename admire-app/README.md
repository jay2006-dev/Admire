# Admire Global Organisation

Welcome to the **Admire Global Organisation** web platform! This is a highly polished, modern marketing and recruitment website built to showcase Admire's training programs, leadership pipeline, and success stories.

## 🚀 Technology Stack

This project leverages a cutting-edge, modern web development stack:
- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Language:** TypeScript
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/) & Custom Vanilla CSS (Design Tokens)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **Icons:** [Lucide React](https://lucide.dev/)

## 🎨 Design Philosophy & Aesthetics

The site is designed with a **premium, dynamic, and immersive user experience** in mind:
- **Glassmorphism:** Extensive use of frosted glass effects, transparent cards, and blurred backdrops to create depth.
- **Dynamic Micro-animations:** Every interaction feels alive. From hover-scaling cards to floating avatars and smooth scroll-reveals.
- **Curated Typography:** Utilizes sleek modern fonts (Inter, Manrope) paired with massive, impactful poster typography (`Bebas Neue`) for hero statements.
- **Vibrant Gradients:** Beautifully tailored dark-mode gradients (soft dark blues, deep purples) blended with high-quality background images.

## 🏗️ Key Features & Sections

We have built and refined several core sections to create a cohesive journey for the user:

### 1. Hero Section
- Features a massive, bold **Bebas Neue** headline with a vibrant orange accent.
- Content is cleanly aligned to the left over a custom, high-tech background image.
- Includes a seamlessly integrated **Video Modal** that pops up with a frosted glass background when users click "Watch Success Stories".

### 2. Programs & Opportunities
- Styled with a sleek **dark-mode aesthetic** using a deep purple overlay.
- Features **transparent, glassmorphic cards** with white text that hover and glow on interaction.

### 3. Success Stories & Leadership
- Built with unique **Blob Avatars**: A custom SVG mesh overlay that creates organic, fluid, colorful blobs behind user initials.
- Leadership cards are fully dynamic, utilizing a data-driven color and shade system for personalized card accents.

### 4. Interactive Navigation & Footer
- **Navbar:** Features an ultra-transparent frosted glass design that elegantly blurs the content beneath it. Includes smooth-scroll routing to all page sections.
- **Footer:** A clean, single-column stacked layout anchored by a beautiful soft dark blue gradient background.

### 5. Contact & WhatsApp Integration
- A dedicated split-grid contact section.
- Features an animated, floating **WhatsApp Action Button** locked to the bottom right of the screen, seamlessly connected to the real WhatsApp API for immediate user engagement.

## 💻 Getting Started Locally

First, install the dependencies (if you haven't already):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
